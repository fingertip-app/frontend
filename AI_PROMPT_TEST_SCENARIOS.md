## 📋 테스트 케이스

### ✅ 1. 구체적 체험명 언급 (원래 문제)

```
AI: 안녕하세요! 어떤 분과 함께 체험을 찾고 계신가요?
👤: 하회탈 만들기

✨ AI: 하회탈 만들기에 관심 있으시군요! 어떤 분위기를 원하세요?...
👤: 같이 만드는 게 좋아요

✨ AI: 하회탈 만들기는 집중력이 필요한 체험이라 몰입감이 좋아요. 예산은...?
```

**검증 포인트:**
- ✅ 사용자가 언급한 "하회탈" 키워드 반영
- ✅ "도자기 물레나 매듭" 같은 무관한 예시 제거

---

### ✅ 2. 지역 + 체험명

```
👤: 전주 한옥마을에서 매듭 공예

✨ vibe: 전북 지역의 매듭 공예에 관심 있으시군요! 어떤 분위기를...
👤: 같이 만드는 게 좋아요

✨ budget: 전북 지역의 매듭 공예는 집중력이 필요한 체험이라 몰입감이 좋아요. 예산은...?
```

**검증 포인트:**
- ✅ "전주" → "전북" 지역 인식
- ✅ 지역 + 체험명 조합 응답

---

### ✅ 3. 시간대 선호도

```
👤: 평일 오전에 도자기 체험

✨ vibe: 도자기 만들기에 관심 있으시군요!...
👤: 같이 만드는 게 좋아요

✨ budget: 도자기 만들기를 평일에 찾으시는군요! 평일엔 비교적 여유롭게 체험할 수 있어요. 예산은...?
👤: 3만원대

✨ duration: 평일 오전이면 여유 있게 즐기실 수 있겠네요! 체험 시간은...?
            도자기는 보통 2~3시간 정도 소요돼요.
```

**검증 포인트:**
- ✅ "평일" 키워드 인식
- ✅ 도자기 특성에 맞는 시간 가이드

---

### ✅ 4. 새로 추가된 체험 (목공예, 가죽공예 등)

```
👤: 친구와 목공예 체험

✨ vibe: 목공예 체험에 관심 있으시군요!...
👤: 가볍게 즐기고 싶어요

✨ budget: 가볍게 즐기면서 할 수 있는 체험들이 많아요. 예산은...?
```

**추가된 키워드:**
- 목공예: 목공, 나무공예, 조각, 목재, 우드
- 가죽공예: 가죽, 레더, 핸드메이드
- 캔들 만들기, 비누 만들기, 향수 만들기
- 전통차 체험, 떡 만들기, 김치 담그기
- 서예, 민화, 수채화
- 등 총 **25+ 체험 카테고리**

---

### ✅ 5. 여러 맥락 조합

```
👤: 주말에 홍대에서 캔들 만들기

✨ vibe: 서울 지역의 캔들 만들기에 관심 있으시군요!...
👤: 가볍게 즐기고 싶어요

✨ budget: 캔들 만들기를 주말에 찾으시는군요! 주말엔 다양한 시간대를 선택할 수 있어요. 예산은...?
👤: 상관없어요

✨ duration: 간단한 공예라면 1~2시간이면 충분해요. 어느 정도 시간을 생각하세요?
```

**검증 포인트:**
- ✅ "홍대" → "서울" 지역 매칭
- ✅ "주말" 시간대 인식
- ✅ "캔들" → 짧은 시간 체험 가이드

---

### ✅ 6. 지역만 언급

```
👤: 경주에서 체험 찾고 있어요

✨ vibe: 경북 지역 체험을 찾으시는군요! 어떤 분위기를...
👤: 전통 문화 체험

✨ budget: 경북 지역의 전통 체험을 찾으시는군요! 예산은...?
```

---

### ✅ 7. Fallback (기본 응답)

```
👤: 뭔가 재밌는 거

✨ vibe: 어떤 분위기를 원하세요?...
👤: 모르겠어요

✨ budget: 좋은 선택이에요! 예산은...? (기본 응답)
```

- [ ] 가죽공예

- [ ] 전통차 체험

### 2. 지역 인식
- [ ] 서울 (홍대, 인사동, 삼청동 등)
- [ ] 전주 → 전북
- [ ] 경주 → 경북
- [ ] 제주

### 3. 시간대 인식
- [ ] 평일 / 주말
- [ ] 오전 / 오후

### 4. 자연스러운 문맥 반영
- [ ] 사용자 언급한 키워드 그대로 사용
- [ ] 무관한 예시 제거
- [ ] 체험 특성에 맞는 시간 가이드
