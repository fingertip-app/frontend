## 현재 상태

진행 상황이 명확하지 않으므로 완료 체크는 보수적으로 비워둔다. 예솔 확인 후 실제 완료 항목을 `[x]`로 갱신한다.

## 책임 범위

`frontend/`는 사용자가 직접 사용하는 Expo/React Native 앱입니다. 로그인, 온보딩, 홈, AI 추천 탭, 카드뉴스, 문화 설명, 체험 상세, 예약, 결제 진입, QR 확인서, 마이페이지 화면을 담당합니다.

## MVP 완료 기준

- 사용자는 앱에서 로그인 여부와 무관하게 AI 추천 탭에서 동행 인원, 관계, 취향을 입력하고 추천 체험을 확인할 수 있다.
- AI 추천/문화 설명 화면은 Spring API 응답의 안내 문구, 출처, 관련 체험을 표시한다.
- 예약 상태, 결제 상태, AI 실패 fallback을 사용자가 이해할 수 있는 화면 상태로 표시한다.
- 6/28 시연용 실제 기기 빌드가 준비된다.
