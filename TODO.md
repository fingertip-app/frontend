# Frontend TODO - 2026-06-28 MVP

## 연동 원칙

- 앱은 `backend/` Spring Boot API만 호출하며 `backend_python/` FastAPI를 직접 호출하지 않는다.
- 2026-06-28 목표는 공개 스토어 출시가 아니라 실제 기기에서 시연 가능한 내부 MVP이다.
- 디자인과 고급 개인화보다 `카드뉴스/검색 -> AI 해설 -> 체험 -> 예약 -> 결제 -> QR` 주 흐름을 우선한다.

## MVP 화면 완료 기준

- 사용자는 온보딩 또는 홈에서 카드뉴스와 AI 검색을 시작할 수 있다.
- AI 해설 화면에서 출처와 추천 체험을 확인하고 상세 화면으로 이동한다.
- 체험 상세에서 일정/인원/메시지를 입력해 예약을 요청한다.
- 승인된 예약에 대해 테스트 결제를 진행하고 QR 확인서를 조회한다.
- 마이페이지에서 예약 상태와 후기/요약을 확인한다.
